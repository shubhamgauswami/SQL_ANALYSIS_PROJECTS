# Amazon Sales Data Analysis Using SQL

## Introduction
The **Amazon Sales Data Analysis** project focuses on extracting meaningful business insights from Amazon product listing data using **SQL queries**. The analysis covers key areas such as **product pricing, discount trends, customer ratings, and review sentiment**, helping businesses understand product performance and optimize pricing strategies.

This project demonstrates practical **SQL skills including filtering, aggregation, joins, CTEs, window functions, and ranking queries** for real-world business analysis.

---

## Dataset Description
The dataset (`amazon_data`) contains detailed product and customer review information, including:

- **product_id** – Unique identifier for each product  
- **product_name** – Product title  
- **category** – Product category  
- **discounted_price** – Price after discount  
- **actual_price** – Original product price  
- **discount_percentage** – Percentage discount applied  
- **rating** – Average customer rating  
- **rating_count** – Number of ratings  
- **review_id** – Unique review identifier  
- **review_title** – Review title  
- **review_content** – Customer review text  
- **product_link** – Product page link  

---

## Objectives
The main objectives of this SQL analysis are:

1. Identify **highly discounted products**
2. Analyze **pricing differences between actual and discounted prices**
3. Find **top-rated and premium products**
4. Extract **customer sentiment from review data**
5. Identify **top discounted products by category**
6. Calculate **sales contribution and product performance metrics**

---

## SQL Queries and Analysis
The project includes multiple SQL queries covering:

### Basic Analysis
- Identify products priced below a specific threshold
- Find products with high discount percentages
- Filter products based on keywords and categories
- Compare actual vs discounted prices

### Customer Review Analysis
- Extract positive and negative review patterns
- Identify highly rated products
- Analyze customer feedback trends

### Advanced SQL Analysis
- Ranking products using **window functions**
- Using **CTEs (Common Table Expressions)** for advanced calculations
- Calculating **percentage contribution of products to category sales**
- Identifying **top discounted products per category**
